R Workshop

UF Marketing Department


Instructor: Taikgun Song    songt@ufl.edu

Friday 3:00-5:00 pm, STZ 200    Join via Zoom


Workshop description:

The workshop is designed to introduces the R programming language to a novice programmer. We will be learning R as a general-purpose programming language and not as a statistics tool. It will NOT be covering how to run various statistical models and packages. At the end of this workshop, students are expected to re-construct data to their needs, learn the basics of obtaining online data, conduct simple text analysis, and perform a graphical display of the results.


Learn how to...

  • read and combine data.
  • making use of web technologies as data source.
  • understand type and structure of data.
  • clean and modify the data.
  • write efficient and reproducible code.
  • data visualization.
  • handling strings and regular expressions. Conduct simple text analytics
